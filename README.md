## Repo map
- `courses/` → each course has **weekly logs** + a **course mini-lab**
- `semester-project/` → 1 **common end-of-semester** project (covers multiple courses)
- `evidence/` → screenshots + exported files (organized per course/week)
- `templates/` → markdown templates used across the repo

## Evidence rules (simple)
- Every week: 1–2 short bullets + 1 proof (screenshot / output)
- End of semester: 1 mini-lab per course + 1 semester project

## Quick start
1) Put the banner: keep `cover.png` in the repo root
